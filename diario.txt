nós aprendemos que o Git é uma ferramenta fundamental para o controle de versões de projetos, permitindo gerenciar e acompanhar as alterações feitas no código ao longo do tempo. entendemos conceitos essenciais como repositórios, commits. Além disso, praticamos comandos básicos, como git init, git add, git commit e git push, compreendendo como sincronizar o trabalho local com plataformas remotas, como o GitHub.
